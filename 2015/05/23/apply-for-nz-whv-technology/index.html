申请新西兰工作旅行签证：技术准备

以下是双马尾菌这两年来网申新西兰工作旅行签证（WHV）所使用的技术方案，这两年来都成功帮自己和别人拿到名额。文中的一些方法、推断可能并不准确。

以下是具体的方法，以及有用程度评分，5 星满分。本文仅提供方法与思路，没有教具体如何填表，涉及到得部分技术也没有详细教程，须读者自行搜索具体做法。

事先练习填表：★★★★★

事先一定要注册个马甲账号练习下填表，熟悉其内容，做到不用多想就能快速复制粘贴预先准备好的资料。可以拿加拿大的作为练习，提交了也没关系可以删除的，不像某些国家例如日本提交了不能删除。练习的时候不要用自己真实护照号等个人信息，也不要用自己主账号，一定要注册一个马甲，这样就算出状况大不了放弃马甲账号就行。

自动填表：★★★

自动填表推荐 firefox 中的 Autofill 3.6 这个插件，先在练习填表的时候填一个页面记录一次所有字段，后面就页面一刷新出来就自动填好。

有一定作用但没有想象中的大，其最大的作用是防止自己填错信息，和你在熟练表格信息的情况下手动填表相比，快的那点时间不会有太大帮助。基本上能够进入填写信息界面就不用太担心超时或者页面一直刷不出来，因为对方服务器已经为你保留了一个连接。

没有自动填表的话，就预先把信息记录在记事本。

代理服务器：★★★★★

通过代理中转来访问移民局的网站，相当于抄小道避免中途的堵车，因为大部分申请人应该没用代理可以绕过中国到新西兰那段相对拥堵的路段。需要注意的时网站入口速度是有限的，这么多人申请无论如何都还是会堵在门口，就算你在新西兰也无法避免，但确实可以大大提高成功申请的概率。双马尾菌申请的时候都会开 3 个浏览器，1 个火狐、1 个 chrome、1 个虚拟机里的火狐，其中一个没有连代理服务器，一直都刷新不出页面。

在各种代理当中，推荐 Shadowsocks，最近很多 VPN 协议都不太稳定、SSH 试过几个服务商最近都经常掉线。Shadowsocks 协议目前比较稳定，和 SSH 类似不是全局流量都走代理，可以按需配置。一定要买付费的服务，而且尽量买最贵的套餐，一个月也就最多 20 元花不了多少钱的。去年使用的 SSH 代理，美国线路；今年用的是 Shadowsocks 代理，日本线路。双马尾菌用的服务商是枫叶主机，比较过几个服务商就这个网页做得最漂亮，细节做得好的服务肯定不会差，事实证明也确实很稳定。

建议要买代理服务的提前一段时间购买并实际试用一下，看长期连接是否稳定，不然中途掉链子就完蛋了。也可以再多买一个备用的。如果想使用枫叶主机的，可以考虑双马尾菌的推广连接：

关闭 javascript：★★★★

在填写健康信息那个页面有十几个要填 yes 的下拉菜单，每选择一个就会刷新一个页面，这样会极大增加服务器的负担，增加自己失败的概率。

这个自动刷新功能是 javascript 语言做的，所以可以到健康信息页面的时候到浏览器关闭 javascript 从而禁止这个行为，火狐和 chrome 都有这个功能。submit 完之后记得打开，因为支付的时候要用到。

多桌面：★★★

windows 可以安装多桌面软件，通过快捷键切换不同桌面，每个桌面放一个浏览器这样会有条理很多。推荐微软官方出得一个小软件 Desktops v2.0，有四个桌面，通过 Alt+1 至 4 来切换。

多浏览器：★★★★★

要想一个人顶几个人就必须用多个浏览器了，推荐 firefox 或者 chrome。一个浏览器程序不能打开多个实例，但可以再打开一个隐私浏览窗口，这样相当于有两个 firefox 和两个 chrome 了，配合多标签页战斗力已经不错。配合上面说到的多桌面，一个桌面放一个浏览器，方便切换不会打乱。

如果想更进一步，可以再开一个虚拟机，里面在运行一个浏览器。不过这样做意义不是特别大了，因为这么多窗口你很难照看过来。

注意信用卡：★★

支付只能使用 visa 或 master 信用卡，如果没有用过境外支付功能最好测试一下，免得由于没有开通等原因支付失败，可以到美国亚马逊买一本便宜的 kindle 电子书试试。还有注意保持余额充足，该还款的提前还款。

查看浏览器调试模式：★

经常输入网址后很长一段时间都没有任何信息出来，等的比较着急。可以在 firefox 中找到 开发者->网络，chrome 也有类似选项，可以显示你和服务器端所有通信的情况。例如你现在输入了网址，当前对方服务器有响应没有、还是说已经响应浏览器正在下载页面，并会显示每项内容打开时间。没有太多实际作用，只是焦急等待时找些东西分散注意力。

类似的还有一直都打不开页面都以为自己网络断了，就去刷刷其他网址，让自己不用害怕网络断了。
