背景

双马尾菌是一个基础知识不太扎实的 Code Monkey，工作几年了维护型的工作居多，开发型的工作有一些但也大部分是锦上添花性质的。考虑到今年年底要重新找工作，所以想巩固一下基础知识。根据自己的实际情况和不多的时间，之前已经复习了软件工程和深入浅出设计模式两本书，接着就是刷 leecode 题目。

对于 SQL 这块，双马尾菌只是在学校学了点基础概念，然后工作中也偶尔用到一些简单查询、建表，所以这个系列文章目标读者也应该是类似背景的，内容会比较啰嗦。这系列 Database 解题文章也算是一个学习笔记，把做题过程中学到的东西写下来。主要特点包含：

  • 每篇文章都包含有如何建立测试表和测试数据，在 MySQL Workbench CE 上运行后就有测试数据了。

  • 包含有解题思路，以及一些需要注意的语法细节、题目陷阱等。

  • 一般答案都包含有多层 subquery，文章会从里到外一层层分步讲解，并给出每一步的运行结果，更方便理解。

  • 如果想到其他答案也会放到另一种答案一节，解释相对会简略一点

接下来打算继续做 Algorithm 类题目，因为题目数量太多所以先做 Database 类，数量少做完以后或许就有信心去挑战数以百计的新题目。
