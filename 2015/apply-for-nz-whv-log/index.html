申请新西兰工作旅行签证：网申实录

By 胖的Panda

发表于 2015-05-23

更新日期 2015-05-24

文章目录

  2. 2. 2014 WHV 实录

以下是双马尾菌这两年来网申新西兰工作旅行签证（WHV）的实录回忆，文中的一些方法、推断可能并不准确。

2015 WHV 实录

2015 年的网申是北京时间 5-19 06：00，双马尾菌是 5：40 左右起来的，然后打开电脑打开代理、打开多桌面、打开多个浏览器。这时没必要做任何事情，提前登陆是没有用的，要做好长期战的心理准备，有吃的准备一下吃一点。

然后 6 点一到就打开登陆界面，打开多个浏览器往各个浏览器都输入登陆的网页，盼着页面出来。如果超时就刷新以下，或者关闭这个标签重新开一个。这段时间是比较煎熬的，也需要很大的耐心。今年感觉比去年花得时间要长，但是经过去年后一点都没有紧张了，虽然有些疑问但也没有任何动摇，相信自己肯定能拿到名额。

经过漫长的等待在 7：30 左右刷出来第一次登陆页面，这时已经过了一个半小时了，马上登陆也成功了。接着准备到中国 WHV 页面开始新建申请，这时却提示退出登陆，难道登陆出错了还是移民局网站有问题？继续尝试重新建申请，但是都没有连接成功一直超时。接下来的一段时间也刷新出一两次登陆页面，但是都没有登陆成功。又看到 QQ 群里面说移民局网站出问题，没有开放中国区的申请，这就可以解释前面新建出错的问题了。

一直刷新，到 8：15 左右 chrome 浏览器又刷出一个登陆页面了，成功登陆并新建申请，顺利进入填写信息阶段。但这个浏览器是备胎没有设置自动填表，只好手动填写，由于比较熟悉表格以及资料都提前写在记事本，也没有太大问题。

接着作为主力的 firefox 浏览器也终于刷新出来登陆页面了，用另外一个马甲账号登陆并开始申请。登陆后页面就比较流畅了，页面出来后就自动填表，然后下一页。中间只出现过一次超时，再打开个新标签页就 OK 了，到填写健康信息时关闭 javascript 防止重复刷新。接着很顺利一直到支付成功，这时已经 8：30 了，就是说奋战了两个半小时。

一些心得：

  • 做好长期作战心理准备，可能需要两三小时甚至更长时间
  • 不要放弃，直到不能新建申请、或者提交时提示已满。就算名额满也要继续刷新，总有些人信用卡没准备好，或者卡在支付环节的

2014 WHV 实录

2014 年的网申是北京时间 7-8 06：00。14 年的状况比较正常，大概 6：30 左右就刷新出来登陆页面。接着大概不到 7：00 就完成了整个申请。
