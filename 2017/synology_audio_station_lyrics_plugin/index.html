Audio Station网易歌词插件

由于国内版权保护越来越严格，加上各大版权联盟之间互相竞争，导致很多歌曲下架或者只能在一个网站独家播放。虽然这是好事，但在大洗牌完成之前普通用户要听歌就麻烦了不少，要多下载几个 APP。另一方面 Hi-Res 音乐大规模发展，在 mora.jp 上能买到很多楼主喜欢的 ACG 音乐的 Hi-Res 版本，一般为 96kHz/24bit 的 FLAC 文件，普通录音源文件差不多就是这个规格，再压缩一下就是 CD。再加上人在国外很多歌曲都仅限中国大陆地区播放，所以楼主又回到了本地听歌的时代，把歌曲放在群晖 Audio Station 上。

这样歌词就成了一个大问题，群晖 Audio Station 自带的歌词插件太弱，歌词都找不到。网络上有个千千静听的插件还不错，早年千千静听用户非常多，也贡献了很多歌词和歌词翻译。但现在早已被收购不再维护，而用户也转移到了腾讯、网易、虾米等等新的网络平台，所以需要有一款新的歌词插件才行。楼主非常喜欢网易云音乐 APP，就决定从这里入手弄一个歌词插件，第一次用 PHP 共花了一个多星期的业余时间才搞定。照惯例先放个声明：

免责声明：本歌词插件仅用于研究机学习 PHP 编程、群晖 NAS 开发、计算机网络的目的，任何用户将其用于非法用途概与作者无关。歌词版权归网易及歌词贡献者、歌曲版权所有者等等所有。

歌词都是网易的网友们上传的，很多都带有中文翻译，在此感谢歌词贡献者。

安装与使用歌词插件

从 GitHub release 下载。有两种口味：netease_org.aum 是清真原味，仅显示歌词原文，适合学霸；netease_trans.aum 是带中文翻译的口味，大众首选。打不开 GitHub 的这里也有度娘盘。

去 Audio Station -> 设置 -> 歌词插件 -> 添加 来加入你喜欢口味的 aum 文件，然后记得打勾开启。

然后当你在 Audio Station 网页或手机端 DS Audio 播放时歌词会自动下载。

如果觉得歌词不够匹配，在 Audio Station 右键单击那首歌，去 歌曲信息 -> 歌词 -> 从网络搜索 获取所有搜索结果。你可以从第二个开始尝试，因为第一个就是自动下载的结果。

下载来看看，觉得不够好的再换回去搜索结果列表，继续换一个。

NOTE：下载完成后 lrc 歌词会缓存在 NAS 的歌曲所在文件夹，可以拷贝到其他地方使用。

歌词插件特色

歌词搜索列表是根据艺术家、标题的相似程度，从高到低排序的，所以结果列表的第一项（Audio Station只自动下载第一项）一般来说是最佳结果。所以如果你的歌曲文件信息是准确的，匹配成功率会非常高。

第二个特色如果安装的是 netease_trans.aum 版本会自动添加中文翻译到行末并用【】标注。网易的歌词原文和翻译是两个独立的 lrc 歌词文件，所以本插件就像网易云音乐 APP 一样把它们拼接起来。

Audio Station智能播放列表

主要讲歌词插件的，顺便加点其他内容吧。播放列表是个麻烦的东西，在一个地方建好了可能过不久又要转移听歌阵地，加上一般喜新厌旧，喜欢新歌单曲循环，所以楼主最常用的列表是最近添加的和 Random100。Audio Station 可以根据你设定的一些条件建立智能播放列表，这样新添加的歌曲只要符合条件就会自动添加进去，方便免维护，需要做的只是确保歌曲信息准确。

例子一《最新一季歌曲》：建立最近 90 天内添加的新歌曲，就是一个动画季度的时间。Audio Station 虽然自带一个最近添加的播放列表，但只能逐个专辑查看、播放，并不好用。

例子二《我的女神们》：包含所有初音未来的歌曲。由于是虚拟歌姬，演出者（艺术家）经常是这样的格式黒うさP feat.初音ミク，所以演出者这项只要包含初音ミク就行，不能精确匹配。后面两项类型（genre）是补充用的。

例子三《梶浦大神》：梶浦由记是幕后大神，仅负责作曲，用智能播放列表就可以把这些歌曲全挑出来。

还有其他条件可以慢慢探索，可以创建你喜欢的智能播放列表。

说说 DS Audio

APP 整体上非常不错，还支持推送到支持 WiFi 音箱播放。也比仅仅通过媒体播放器 DLNA/UPnP 协议播放的功能要好很多，特别 iOS 上那些 DLNA 播放器 APP 要收费而且 UI 巨丑。最难的的是 Windows 平台也有对应 APP。

歌词问题解决后，还有个很讨厌的问题是个别音乐的专辑封面死活读取不出来，尝试更换封面等等，完全没发现规律。后来放弃了自制了个 2233 娘做完默认封面，总比白板好。

歌曲信息编辑

有人可能会说歌曲信息不准确怎么办？在复制音乐进去 NAS 之前用编辑软件 Mp3Tag 先编集一下信息和专辑封面就好。Mp3Tag 是一款依靠自愿捐款的免费软件，好用且强大，有中文版，可以批量修改歌曲信息。格式支持非常好，包括 ALAC、FLAC、DSD 等都支持。原生是 Windows 平台，MacOS／Linux 字体委屈一点也有 wine 版，毕竟这两个平台软件少。

想要批量编辑的，把需要编辑歌曲全部拖进去，选中所有歌曲，例如想要修改所有歌曲的类型（genre）为 Anime，那么编辑红色框那个类型栏，点击保存就行。其他无需改变的字段它晖显示为 <keep>，表示保留原值。包括专辑封面图片都可以批量更改。

现在中档以上手机屏幕分辨率都是 1080p 起步，旗舰机都是 2k 以上，所以专辑封面也是时候升级了。很多下载的歌曲还是史前的 500px 规格，建议找更高分辨率的，至少 1500px 吧。
